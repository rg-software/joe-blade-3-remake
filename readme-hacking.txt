Release
Полная поставка игры. Нужна прежде всего для того, чтобы понять, какие файлы требуются для запуска :)

LevelOptimizer
Уровни рисуются с помощью редактора уровней (что логично). Готовые уровни имеют расширение .lev
В игре же требуются не уровни, а файлы отдельных комнат (.jb3). LevelOptimizer считывает все lev-файлы из текущего каталога, разбивает их на отдельные файлы и кладёт в папку Rooms. Кроме того, он создаёт индексные файлы joeblade3.rooms и joeblade3.tiles
Для каждого lev-файла требуется вручную создать start-файл, в котором прописываются координаты стартовой комнаты (обычно комнаты с лифтом).

Resources
Собрание всех ресурсов игры плюс обобщённый генератор ресурсных файлов (MakeResFile) плюс редактор уровней. После запуска надо просто набрать имя каталога с ресурсами. В результате будут созданы ресурсный файл и файл-заголовок.

JoeBlade
Собственно исходник. Все ресурсы уже подключены (в виде ресурсных файлов).

Libs
Текущие версии используемых библиотек (следующие версии могут быть уже несовместимыми ;) )

Additional
Дополнительные файлы, использовавшиеся при создании игры.
